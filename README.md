# Real-Time-Chat-History-Mensajeria-
● Por qué NoSQL: Una sala de chat guarda los últimos 50 mensajes directamente dentro del documento de la sala para cargarlos rápido al abrir la app.
● Reto: Patrón de "Bucketing" (agrupar mensajes por bloques de tiempo) y límites de tamaño del documento BSON.
